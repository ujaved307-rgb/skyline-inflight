# SKYLINE — In-Flight Infotainment (standalone concept)

A reimagined seat-back experience. Tablet canvas 1024×600.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
```bash
npx vercel
```
Press Enter through the prompts. You'll get a public URL like
`https://skyline-inflight-xxxx.vercel.app`.
